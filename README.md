# Decision-Making Statements in Java

During my Java learning journey, I learned about decision-making statements. These statements help a program make decisions and execute different blocks of code based on conditions.Java decision statements evaluate true/false conditions to guide your program down the right logical pathway. They act as crossroads in your coding journey, choosing exactly which instructions to execute or skip.

// if statement
if (condition) {
    // code
}

// if-else statement
if (condition) {
    // code
} else {
    // code
}

// if-else-if ladder
if (condition1) {
    // code
} else if (condition2) {
    // code
} else {
    // code
}

// Nested if
if (condition1) {
    if (condition2) {
        // code
    }
}

// switch statement
switch (expression) {
    case value1:
        // code
        break;

    case value2:
        // code
        break;

    default:
        // code
}

## What I Learned

* Used `if` to execute code when a condition is true.
* Used `if-else` to choose between two options.
* Used `if-else-if` for multiple conditions.
* Learned how nested `if` statements work.
* Used the `switch` statement to simplify multiple choices.

This topic helped me understand how Java programs make decisions and control the flow of execution. It is one of the fundamental concepts required for building real-world applications.
